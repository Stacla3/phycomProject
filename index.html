LED-Based Mood Light with Color Fade

Abstract

โครงการนี้มีเป้าหมายสร้างโคมไฟที่เปลี่ยนสีแบบไล่ระดับอัตโนมัติหรือควบคุมผ่านแอป Blynk โดยใช้อุปกรณ์ Arduino, LED RGB หรือ RGB Strip, Breadboard, เซ็นเซอร์ตรวจจับการเคลื่อนไหว (PIR) และเซ็นเซอร์วัดอุณหภูมิและความชื้น (DHT) เพื่อให้ LED เปลี่ยนสีไล่ระดับอย่างนุ่มนวล สร้างบรรยากาศที่ผ่อนคลาย พร้อมเชื่อมต่ออุปกรณ์เข้ากับบอร์ดควบคุม

input

  • การปรับค่าจากDashBoard ผ่านArduinoCloud ซึ่งจะให้ค่าออกมาเป็น RGB เพื่อเป็นinput ให้กับหลอดไฟ
  • อุณหภูมิที่วัดได้จะเป็น input ในการปรับสีของหลอดไฟไปตามอุณหภูมิที่วัดได้
